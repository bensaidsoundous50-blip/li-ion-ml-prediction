# Machine Learning for Li-ion Battery Materials

## Objectif
Prédire la capacité électrochimique de cathodes Li-ion à partir de leur composition chimique.

## Méthode
Modèle de régression Random Forest reliant composition (Li, Ni, Mn, Co) et capacité.

## Outils
Python, pandas, scikit-learn, matplotlib

## Résultat
Modèle simple permettant d’illustrer la relation composition-performance.
